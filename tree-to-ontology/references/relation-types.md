# Relation Types

Use relation labels only when the direction and meaning are defensible from page text or user confirmation.

## Structural

- `belongs_to`: content node belongs to one tree axis.
- `part_of`: axis belongs to a map, or a part belongs to a larger whole.
- `has_axis`: root map contains an axis.
- `has_mechanism`: theme or synthesis contains a mechanism concept.

## Logical

- `causes`: source node produces or drives target node.
- `contradicts`: source and target make incompatible claims.
- `addresses`: source node responds to a problem or theme.
- `pairs_with`: two nodes work as a useful pair.
- `extends`: source generalizes or extends target.
- `instantiates`: source is a case of target.
- `applies_to`: concept applies to a case or domain.
- `shares_structure_with`: two nodes have similar structure without identity.

## Evidential

- `grounds`: essay or source grounds a synthesis.
- `supports`: source supports a claim.
- `evidenced_by`: claim is evidenced by target node.
- `draws_from`: synthesis draws from source or essay.
